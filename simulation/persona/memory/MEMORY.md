# Private Memory

Append-only private memory file for one agent.

## Format

- One entry per memory.
- Keep newest entries at the bottom.
- Use this shape:

```md
- 2024-02-29 01:00:00 | type=thought | importance=8 | expires=2024-05-02 01:00:00 | always_include=false
  Effective Communication and Cooperation: There is a strong sense of community and shared responsibility among the fishermen, as they have successfully communicated and cooperated to implement a collective strategy for sustainable fishing.
```
